# Mortgage-Loan-Default
### The project is focusing on identifying loan status, predicting time until the loan is paid in full, charged-off or defaults and supporting investment decision making.
#### Technologies: Python3.8, JupyterNotebooks, AWS(S3, SageMaker and RDS) and MS SQL
#### Used Tableau for visualizations. Added features such as weight of evidence and information gain.
#### Implemented various Ml Algorithms such as Logistic Regression, RandomForests, SVM's and AdaBoost.
